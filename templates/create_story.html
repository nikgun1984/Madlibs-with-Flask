Instructions

Using the form below, please adhere to the following rules:

  1. 1. Write an entire story inside the text box.
  2. 2. For words you want to omit, add square brackets [ ]
  3. EXAMPLE: Once upon a time there lived a(n) [ ]
  4. 3. Add descriptive word for omitted word inside the square brackets.
  5. EXAMPLE: Once upon a time there lived a [person] NOTE: It can be any descriptive word.
  6. 4. Submit the form and select your newly created story in "CHOOSE STORY" menu.
