Instructions

Using the form below, please adhere to the following rules:

  1. 1. Enter the title of your story
  2. 2. Write an entire story inside the text box.
  3. 3. For words you want to omit, add curly brackets { }
  4. EXAMPLE: Once upon a time there lived a(n) { }
  5. 4. Add descriptive word for omitted word inside the curly brackets.
  6. EXAMPLE: Once upon a time there lived a {person} NOTE: It can be any descriptive word.
  7. 5. Submit the form and select your newly created story in "CHOOSE STORY" menu.

ENJOY!!!
